## Statement
```
Given an integer, print "odd" if it's odd and print "even" otherwise.
```
## Sample Input1
```
5
```
## Sample Input2
```
6
```
## Code
```
#Read an Integer:
a = int(input())
#Print a value is odd or even:
if(a%2==0):
  print("even")
else:
  print("odd")
```
## Sample Output1
```
odd
```
## Sample Output2
```
even
```
